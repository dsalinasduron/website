Overview

A heap is a complete binary tree whose nodes contain Comparable objects. In a maxheap (maximum heap), the value of a parent is >= each of its children, however there exists no requirement as to the relationship of the children.




Heap Interface

- MaxHeapInterface.java

It is important to note that the interface does not specify whether the implementation is a max or min heap.

Uses of a Heap

A maxheap can be used to implement a priority queue. The item at the top of the heap represents the item with the highest priority.
