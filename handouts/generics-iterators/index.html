Generics

The following program demonstrates how to use Java generics:

- UseArrayList.java

- Item.java (items we place in the ArrayList)

Notice we need to type-cast what returns from the get() method of the ArrayList

contents = (Item)list.get(i);

Java generics allows a generic data type to be stored in a data structure, without having to typecast the return value. It also does compile time checking to ensure we in use the data structure correctly.

Without generics

List list = new ArrayList();

With generics

List<Item> list = new ArrayList<Item>();

This indicates that Item is the only type of object that can go into list .
