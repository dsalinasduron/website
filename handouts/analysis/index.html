Standard Growth Rate Functions



Orders of growth refers to how quickly the algoithm slows down as the size of the input increases. (Intuitively this should make sense.)

Fast growth is bad - slow growth is good.

In general

1 < log n < n < n log n < n^2 < n^3 < 2^n < n!

Big-Oh Notation

f(n) <= c * g(n) for all n >= n0
