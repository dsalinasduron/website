Dictionaries

A dictionary is a data structure that allows us to look up the value of a keyword.

Objectives

- Provide an overview of dictionaries and their operations

- Examine applications that use dictionaries.


Overview

A dictionary is also called a map, table, or associative array. It works as follows,

there is a unique key (or keyword)

and a value that is associated with that key.

Therefore, we associate [key:value] pairs.

Examples:

Key
 Value
Social Security Number
 Person
ISBN Number
 Book
Automotive VIN
 Car
GPS Coordinates
 Location on Earth
Username
 Password
?
 ?

The key must be unique, the value need not be. For example, there may be multiple books with the same title, many cars of the same make/model, etc.
