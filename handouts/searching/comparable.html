Notes on Using Comparable

Java checks for object equality using the equals() method which returns a boolean indicating if two objects are equal to one another. However, equals() is not appropriate if we want to check how to objects are related to one another, such as > or <.

The Comparable interface in Java allows determining how two objects are related to one another:

Object o1;
Object o2;

o1.compareTo(o2) returns (1) a negative integer value if o1 is < o2, (2) 0 of o1 equals o2, and (3) a positive integer value if o1 > o2.
