CMPT 202 
Lab 4: Linked Lists



Objectives

The objectives of this lab are:

    * To review using linked lists
    * To implementation several methods for a linked list
    * To apply some principles of programming and software engineering 

Before Proceeding

Review the Programming Grading Guidelines for ideas as how to properly write Java programs.

The Lab

Java Files:

List.java specifies the interface for a linked list.

TestLinkedList.java is a set of unit tests that test an implementation of the List interface.

LinkedList.java is a partially-completed implementation of the List interface.

Complete the remainder of LinkedList.java so that it passes all unit tests. The methods that need to be completed are identified with a TO DO comment.

A few of the easier methods to first  complete are isEmpty() and clear() . It is suggested that you then complete get() .

It is recommended that you first complete both add() methods, followed by both remove() methods. It is important to identify the special case of when you are adding or removing to the head of the list. When this occurs, you are modifying the head instance data.
