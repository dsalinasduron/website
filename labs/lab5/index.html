Part I - Write a Program That Uses Generics and Iterators

Before you proceed with implementing a class that provides both generics as well as iterators, you will first write a simple program that uses generics and iterators.

The ArrayBag class shown below provides both generics as well as an iterator.

- Bag.java (the interface)

- ArrayBag.java (the implementation)

Write a simple program named UseArrayBag.java that creates an ArrayBag that contains String objects. You can write everything in the main() method for the UseArrayBag class.

Add the following String objects to your bag:

"Apple"
"Banana"
"Cherry"
"Donut"
"Eggs"

Be sure to use generics.

Next, get an iterator from your bag (by calling the iterator() method) and print out the objects from the iteration using a while loop. Your output should of course appear as

apples
bananas
cherry
donuts
eggs


It may be useful to review the UseIterator.java class before proceeding as your UseArrayBag  program will use a very similar approach.

Part III - Implement a Set using an ArrayList

Implement the Set interface using an ArrayList from the Java API. Name this implementation ListSet .

This should be relatively easy to complete as most of the methods in the Set interface can be delegated to equivalent methods available from ArrayList.

Test your implementation by ensuring you pass the following unit tests:

- TestListSet.java

Once you pass the above unit tests, the following code example demonstrates abstraction:

       Set<String> s1 = new ArraySet<String>();
   Set<String> s2 = new ListSet<String>();
       
   s1.add("apples");
   s1.add("cherries");
   s1.add("eggplant");
       
   s2.add("brownies");
   s2.add("donuts");
       
   Set<String> s3 = s1.union(s2);
       
  
This code example illustrates how you can perform a union of two different implementations of the Set interface.

The complete code example:

- UnionDifferentSets.java

Run this program and make sure the program output matches what you expect.
